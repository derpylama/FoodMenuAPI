Dokumentation

Skicka till /index.php?week=weeknr

weeknr = det vecko nummer som du vill få menu till.

veckan kan max vara 52 och minst 1

formatet som datan skickas tillbaka

exempel:/index.php?week=2

"2": { "menu": ["Risotto med oxfilé", "Kåldolmar", "Chili con carne", "Pasta puttanesca", "Sill och potatis"],"vegeterianMenu": ["Miso soppa", "Rödbetsbiffar", "Vegetariska tacos", "Tofu satay", "Kikärtsbiffar"]}

Där menu innehåller de fem olika maträtter som serveras för alla vardagar. (mån-fre) för att komma åt menu skriv "data["menu"]" får du tag på en array med vilka maträtter som är den listan.

För att få en specifik dag välj ut den ur listan som du får för hela veckan.
